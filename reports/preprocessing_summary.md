# Preprocessing Summary

## Цель этапа

Подготовить исходный датасет IMDb Movies к дальнейшему разведочному анализу.

## Выполненные операции

1. Проведен первичный обзор данных:
   - просмотр первых строк;
   - анализ размера таблицы;
   - проверка типов данных;
   - оценка пропусков.

2. Проверены дубликаты:
   - полные дубликаты;
   - дубликаты по названию фильма.

3. Обработан признак `runtime`:
   - удалены текстовые элементы вроде `min`;
   - значения приведены к числовому типу;
   - выявлены некорректные или отсутствующие значения.

4. Обработан признак `gross_earn`:
   - удалены символы `$` и `M`;
   - значения приведены к числовому типу;
   - сохранены пропуски там, где кассовые сборы отсутствуют.

5. Данные отсортированы по рейтингу.

6. Итоговый датасет сохранен в форматах:
   - `movies_clean.csv`;
   - `movies_clean.xlsx`.

## Результат

Получен подготовленный датасет для дальнейшего EDA-анализа.
