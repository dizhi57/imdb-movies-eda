# IMDb Movies Dataset — Exploratory Data Analysis

## Цель проекта
Провести разведочный анализ датасета IMDb Movies.

## Что сделано
- изучена структура данных;
- выявлены пропуски и нулевые значения;
- проанализированы распределения кассовых сборов и продолжительности;
- найдены выбросы;
- построены histogram, boxplot, scatterplot;
- описаны основные закономерности.

## Стек
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook.

## Результат
Подготовлен EDA-отчет с описанием структуры данных, проблем качества и содержательных выводов.
