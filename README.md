# IMDb Movies Dataset — Data Preprocessing & Exploratory Data Analysis

## Цель проекта

Провести подготовку и разведочный анализ датасета IMDb Movies, содержащего данные о 5000 фильмах с высоким рейтингом. Проект включает два этапа: предобработку данных и последующий EDA-анализ.

## Данные

Источник: IMDb Movies Dataset, Kaggle.  
Датасет содержит сведения о фильмах: название, рейтинг, длительность, жанр, режиссер, актеры, количество голосов, кассовые сборы и другие признаки.

## Структура проекта

- `notebooks/01_imdb_movies_preprocessing.ipynb` — обзор и предобработка данных;
- `notebooks/02_imdb_movies_eda.ipynb` — разведочный анализ данных;
- `data/processed/movies_clean.csv` — очищенный датасет;
- `images/` — визуализации из анализа.

## Этап 1. Предобработка данных

В ноутбуке `01_imdb_movies_preprocessing.ipynb` выполнены:

- загрузка исходного датасета;
- обзор структуры таблицы;
- проверка размера датасета;
- проверка полных дубликатов;
- проверка дубликатов по названию фильма;
- анализ пропусков;
- преобразование признака `runtime` к числовому типу;
- преобразование признака `gross_earn` к числовому типу;
- сортировка данных по рейтингу;
- сброс индекса;
- сохранение очищенного датасета в CSV/XLSX.

## Этап 2. Разведочный анализ данных

В ноутбуке `02_imdb_movies_eda.ipynb` выполнены:

- анализ распределений количественных признаков;
- анализ кассовых сборов;
- анализ продолжительности фильмов;
- анализ категориальных признаков;
- поиск выбросов;
- построение визуализаций;
- формулировка аналитических выводов.

## Стек

Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook.

## Результат

Подготовлен очищенный датасет и EDA-отчет с описанием структуры данных, проблем качества, выбросов и содержательных взаимосвязей.
